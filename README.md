Cosmic Postal Chess
=======

Synopsis
--------
A way for me to play my grandfather chess from anywhere in the world.

Usage
-----
<pre>npm start</pre>
or
<pre>node server.js</pre>

Features
--------
* node.js application server
* Pub-sub messaging with faye.js
* Movable chess pieces
* Basic castling
* Does not check legality of moves
